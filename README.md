# Logistics Dashboard Test

A local Next.js dashboard test app for trying the sign-up, login, and dashboard flow without needing a live backend setup.

## What You Can Do

- Create a local account through the sign-up flow
- Log in with the account you created
- Access the dashboard after authentication
- Exercise the app with simulated server behavior so the core flow can be tested end to end

## Getting Started

Run the app locally:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Notes

- The project is set up for local testing and development.
- Auth flows and dashboard interactions are intended to work inside the app without depending on an external production deployment.
- If you change the environment variables, restart the app so the new values are picked up.
